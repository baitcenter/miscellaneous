# vue-demo

## 无渲染组件

### Slot

作用：

1. 数据
2. 动作
3. 绑定

### 场景

- 你正在构建一个库，并且希望使用者能够更轻易地自定义组件的外观（样式效果）
- 你项目中有多个组件，其行为或功能非常相似，但布局不同
